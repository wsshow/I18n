# I18n

golang 简单的多语言

## example

```json
[
  {
    "id": "core program was not found",
    "langs": {
      "en": "core program was not found",
      "zh": "未找到核心程序"
    }
  },
  {
    "id": "not found core module: %s",
    "langs": {
      "en": "not found core module: %s",
      "zh": "未找到核心模块：%s"
    }
  },
  {
    "id": "current goroutine num: %d",
    "langs": {
      "en": "current goroutine num: %d",
      "zh": "当前协程数：%d"
    }
  }
]
```
